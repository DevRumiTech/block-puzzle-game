A fast, responsive block puzzle game playable on mobile and desktop. Works offline.

---

Features

* Smooth keyboard and touch controls
* Mobile and desktop friendly
* Sound effects with mute toggle
* Hold and next piece preview
* Scoring, levels, and high score save
* Pause and resume support
* Fully offline (no dependencies)

---

Controls

Keyboard

* ← → : Move
* ↓ : Soft drop
* ↑ : Rotate
* Z : Rotate (CCW)
* Space : Hard drop
* C : Hold piece
* P : Pause
* R : Restart
* M : Mute

Mobile

* Swipe left/right → move
* Swipe down → soft drop
* Swipe up → rotate
* Tap → hard drop

---

Play

Open the game in your browser or use GitHub Pages to play online.
